# Sales Prediction Using Python

## Objective
Predict product sales based on advertising expenditure.

## Dataset
Advertising Dataset

Features:
- TV
- Radio
- Newspaper

Target:
- Sales

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Algorithm
Linear Regression

## Results
- Mean Absolute Error: 1.27
- R² Score: 0.91
