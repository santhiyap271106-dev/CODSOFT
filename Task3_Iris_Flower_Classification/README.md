# Iris Flower Classification

## Objective
Build a Machine Learning model to classify Iris flowers into:
- Setosa
- Versicolor
- Virginica

## Dataset
Iris Dataset (150 samples)

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Algorithm
Random Forest Classifier

## Results
- Accuracy: 100%
- Successfully classified all flower species
