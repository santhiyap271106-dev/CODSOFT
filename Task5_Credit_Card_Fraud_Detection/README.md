# Credit Card Fraud Detection

## Objective
Detect fraudulent credit card transactions using Machine Learning.

## Dataset
Credit Card Fraud Detection Dataset

Features:
- Time
- V1 to V28
- Amount

Target:
- Class
  - 0 = Genuine Transaction
  - 1 = Fraudulent Transaction

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Algorithm
Logistic Regression

## Data Preprocessing
- Checked missing values
- Balanced the dataset using random sampling
- Split data into training and testing sets

## Results
- Accuracy: 94.92%
- Precision: 95%
- Recall: 95%
- F1 Score: 95%
